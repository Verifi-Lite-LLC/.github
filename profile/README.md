## ✨ Features

- **Document Verification** — 190+ countries
- **Face Verification** — Passive + Active Liveness, Deepfake Detection
- **AML Screening** — OFAC, PEP, Sanctions, Adverse Media + Continuous Monitoring
- **On-Chain Identity** — Link real IDs to wallet addresses (EVM)
- **Non-custodial Wallets** — Generate wallets without holding keys
- **KYC Widget + SDK** — Web & Mobile (React, Flutter, etc.)
- **REST API + Webhooks**
- **Sandbox Mode** for testing
